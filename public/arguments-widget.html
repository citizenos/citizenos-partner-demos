CitizenOS Partner Demos

Arguments Widget

Widget for embedding CitizenOS arguments system into your webapp.

Implementation notes:

  • Assumes Partner to keep the Partner content id to Topic id mapping to know what Topic id to use to open the IFRAME

Example

        // Load inside your own element with id 'myCitizenOSWidgetElementId'
        var argumentsWidget1 = window.CITIZENOS.widgets.Argument('f21b10b5-9147-4b12-b6f1-1d2380ff752b', 'myCitizenOSWidgetElementId');
        // Load as a last element in body with id auto generated
        var argumentsWidget2 = window.CITIZENOS.widgets.Argument('f21b10b5-9147-4b12-b6f1-1d2380ff752b');

NOTE: Everything inside green is served by CitizenOS
