CitizenOS Partner Demos

Arguments Widget

Widget for embedding CitizenOS arguments system into your webapp.

Example

Add following script to your page: 
        <script src="https://citizenos-citizenosfe-web-test.herokuapp.com/js/widgets.arguments.js"></script>
Load the widget: 
        // Load with Partner entity ID and Partner ID
        // Load inside your own element with id 'myCitizenOSWidgetElementId'.
        var argumentsWidget1 = window.CITIZENOS.widgets.Argument('et', 14801, '0c580018-ed74-42cb-b0e4-8df654ddd42c', 'myCitizenOSWidgetElementId');
        // Load as a last element in body with id auto generated
        var argumentsWidget2 = window.CITIZENOS.widgets.Argument('ru', '87287210-e215-4e98-8630-5b0ef3b2e5a8');

NOTE: Everything inside green is served by CitizenOS
